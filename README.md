# GlobalPolls

A compilation of party opinion polls, categorised by continent, region and country. The graphics are made using Python, specifically `matplotlib` and `numpy`.

# Features
- Compare current trends between political parties.
- Visualise change in party support compared to previous elections.

# Installation
To install dependencies run:
```python
pip install -r requirements.txt
```
You can then follow the instructions in `Makefile`.

# Usage
- Run commands such as `make greece`, as shown in `Makefile`.
- You can modify percentages based on your estimations.
